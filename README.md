# **CryptoVault**

**CryptoVault** - Безопасное хранение паролей локально на вашем компьютере

## Что себя представляет

- Безопасное хранилище ваших данных с криптографией


### Безопасность

- Мы используем криптографиеческие технологии для безопасности данных

- Технологии:

| Технология |
|------------|
| Fernet |
| PBKDF2HMAC |
| SHA256 |
| Base64 |
| Secrets (Python Libraly) |

- Ваши данные находятся в безопасности, подробнее:

| Что хранится | Как хранится (Какие технологии) |
|--------------|---------------------------------|
| Пароль для Регистрации/Входа | `SHA256 + Secrets + Base64` |
| Ключ шифрования Fernet пользователя | `Base64` |
| Данные о информации логина | `Fernet` |

### Примечания

- Данные которые выводите для сохранения остаются локально на вашем компьютере в файле .db. Разработчик не имеет доступ к базе данных

- Потеряв пароль от аккаунт вы не сможете войти в аккаунт! Вы в итоге потеряет данные! Восставновить или взломать невозможно!
