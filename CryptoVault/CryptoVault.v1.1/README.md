# **CryptoVault**

***CryptoVault*** - Безопасное хранилище паролей с криптографией

## **Что нового добавили в CryptoVault v1.1**

### **Функции**

#### **Функция `Экспорт паролей в CSV/JSON`**

- Данная функция позволяет вам - сделать экспорт ваших паролей в CSV/JSON (В зашифрованном виде)

- Подробнее:

| Формат | Безопасность | Предупреждения |
|--------|--------------|----------------|
| CSV (Таблицы) | Экспортирует в CSV в расшифрованном виде | Если нет данных - нет экспорта |
| JSON (Ключ-Значение) | Экспортирует в JSON в расшифрованном виде | Если нет данных - нет экспорта |

- Пример для CSV:

```csv
Сайт,URL,Логин,Пароль,Категория,Заметки,Дата создания
MAX,не знаю,alexx,random,messanger,НЕ НАСТОЯЩИЙ,2026-04-14 18:11:00
```

- Пример для JSON

```json
[
  {
    "website": "MAX",
    "url": "не знаю",
    "username": "alexx",
    "password": "random",
    "category": "messanger",
    "notes": "НЕ НАСТОЯЩИЙ",
    "created_at": "2026-04-14 18:11:00"
  }
]
```

#### **Функция `Изменение информации логина`**

- Данная функция позволяет вам изменять информацию о логине

- Подробнее:

| Данные (Можно ли изменить) | 
|----------------------------|
| Сайт - да |
| URL - нет |
| Логин - да |
| Пароль - да |
| Категория - нет |
| Заметки - нет | 

> **Предупреждение: изменив данные, вы не сможете увидеть прошлые данные**

## Примечания

- Данные которые выводите для сохранения остаются локально на вашем компьютере в файле .db. Разработчик не имеет доступ к базе данных

- Потеряв пароль от аккаунт вы не сможете войти в аккаунт! Вы в итоге потеряет данные! Восставновить или взломать невозможно!
